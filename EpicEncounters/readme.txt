Мод Epic Encounters, брать тут: https://www.nexusmods.com/divinityoriginalsin/mods/103

Для работы с модом понадобится утилита ExportTool для распаковки и упаковки игровых архивов. Брать тут: https://s3.eu-central-1.amazonaws.com/nb-stor/dos/ExportTool/ExportTool-latest.zip

Для перевода мода нужно сперва распаковать файл epic_encounters_071a986c-9bfa-425e-ac72-7e26177c08f6.pak из папки %USERPROFILE%\Documents\Larian Studios\Divinity Original Sin Enhanced Edition\Mods в папку %Divinity%\Data так, чтобы в ней оказались папки Public и Mods. Сам pak-файл удаляем, он больше не нужен.

Текстовые строки лежат в %Divinity%\Data\Mods\Epic_Encounters_071a986c-9bfa-425e-ac72-7e26177c08f6\Localization. Распаковываем *.lsb в *.lsx, правим, запаковываем обратно туда же.

Текущая версия мода: 1.1.9.5c.
